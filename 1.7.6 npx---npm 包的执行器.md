# npx


## INSTALL
```
npm install -g npx
```


## usage

举例：使用create-react-app创建一个react项目。

- 老方法：
```
npm install -g create-react-app
create-react-app my-app
```
- npx 方式：
```
npx create-react-app my-app

这条命令会临时安装 create-react-app 包，命令完成后create-react-app 会删掉，不会出现在 global 中。下次再执行，还是会重新临时安装。
```
