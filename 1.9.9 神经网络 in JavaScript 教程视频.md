# 神经网络 in JavaScript 教程视频
>使用Brain.js构建神经网络的有趣实用介绍


## 1.介绍

```
const net = new brain.NeuralNetwork({ hiddenLayers: [3] });

const trainingData = [
  { input: [0, 0], output: [0] },
  { input: [0, 1], output: [1] },
  { input: [1, 0], output: [1] },
  { input: [1, 1], output: [0] }
];

net.train(trainingData);

console.log(net.run([0, 0]));
```
